This is a very basic ABC Bank Website to show how Invoid Web Widget will work.

The widget can be added as small component. It will take full width and height of parent element. You can add styles to class "root" to apply your styles.

We can also pass data-attributes to widget to change Document Name or Background Color according to your use Case.

Or Just open another window and then start the proccess.

This webiste was created using plain HTML and CSS to show widget can be applied to any website. The widget code is written in react.
